1 Exercise 1: Start Chatting with Your AI

Now let's load your model and start using it.

Starting a chat:
  1. In LM Studio, click the Chat icon (speech bubble) on the left sidebar
  2. At the top, click Select a model to load
  3. Choose the model you just downloaded
  4. Wait a moment while it loads (you'll see "Model loaded" when ready)
  5. Type your first message in the chat box at the bottom

Quick Test:

Try this simple prompt to test everything is working:

Explain in simple terms: What's the difference between AI running on my laptop versus AI running in the cloud?

You should get a clear, helpful response in a few seconds. If you do, you're all set! If not, ask for help.

What to expect from LFM2: Responses will be quick and clear, though simpler than larger models like GPT-4 or Claude. That's perfectly fine for learning - the trade-off for speed and working on any laptop is worth it!

2 Exercise 2: Make It Fun with System Prompts

Here's where things get interesting. You can give your AI a personality by using a system prompt. This tells the AI who to "be" and how to respond.

Why LFM2 is perfect for this: This tiny model is brilliant for learning about system prompts. The characters might be simpler than with larger models, but you'll clearly see how system prompts shape AI behaviour!
How to set a system prompt in LM Studio:
  1. Switch to User mode (bottom of screen)
  2. In the Chat window, look at the top right for the "Add Instructions" button
  3. Click it - you'll see "Provide custom instructions or guidelines to the AI"
  4. Type or paste your character description there
  5. Start chatting - your AI will now respond in character!

Exercise 3: Explain It Like I'm 5 (ELI5)

Scenario: You need to understand a complex topic quickly, or explain something difficult in simple terms.

What to do:
  1. Start a new chat (click the + button in LM Studio)
  2. First, try WITHOUT a system prompt - just paste the prompt below
  3. Then start ANOTHER new chat and try WITH the system prompt
  4. Compare the difference!

Try WITHOUT system prompt first:

Teach me about quantum computing like I am 5 (ELI5)

Now try WITH this system prompt:

Click the "Add Instructions" button at the top right and paste this:

You explain complex topics to complete beginners. Use simple analogies, everyday examples, and no jargon. Break ideas into simple components.

Now paste the same prompt: "Teach me about quantum computing like I am 5 (ELI5)"

What you'll notice: The version WITH the system prompt should give you a more focused, simpler explanation with better analogies. That's the power of system prompts - they set the AI's expertise and communication style!

Why this matters: Perfect for learning new technical skills, understanding complex processes, or preparing to explain difficult concepts to colleagues with different expertise levels.

Try with your own topics: Replace the topic with anything you need to learn - "blockchain," "GDPR regulations," "machine learning," "how mortgages work," etc.

üí° Performance Tip: Start New Chats Regularly

Your AI works best with a fresh start! Here's why:

  • Prevents confusion: Long conversations can make the AI lose focus or contradict itself
  • Saves memory: Each message uses RAM - new chats free up resources
  • Better responses: Starting fresh gives the AI a clean slate for each new task

Rule of thumb: Click the + (New Chat) button when switching topics or after 10-15 exchanges. Think of it like clearing your desk before starting a new project!

Tips for Getting Better Results

0. Start New Chats Regularly (Most Important!)

Click the + button to start fresh when:

  • Switching to a different topic or task
  • After 10-15 message exchanges
  • If responses start feeling "off" or confused
  • Before important work tasks

Long conversations drain memory and confuse the AI. Fresh chats = better responses!

1. Be Specific

Instead of: "Write about climate change"

Try: "Write a 200-word summary of the main causes of climate change for a general audience"

2. Set the Context

Tell the AI what role to take: "You are a technical writer..." or "You are a project manager..."

3. Specify the Format

Ask for bullet points, paragraphs, tables, or other specific formats when needed.

4. Iterate

If the first response isn't quite right, refine your prompt or ask follow-up questions.

Remember: AI Can Make Mistakes

Always fact-check important information. Local AI is a tool to help you work faster, not a replacement for your expertise.

When to Use Local AI vs Cloud AI

üîí Privacy & Security Advantages of Local AI

Your data stays on your laptop:

  • No IP address leaks: Cloud AI services log your IP address with every request. Local AI never phones home.
  • Air-gapped operation: Local AI has no internet connection - it can't accidentally send your data anywhere.
  • No search keyword leaks: Cloud AI often searches the web to answer questions, potentially revealing what you're working on through search queries. Local AI can't search, so nothing leaks.
  • Truly private: Your sensitive documents, personal data, and confidential work never leave your device.

‚ö° Practical Advantages of Local AI

  • Works without WiFi: On trains, planes, in the field - anywhere your laptop goes, your AI goes too.
  • Lower energy usage: No data centres, no transmission overhead - just your laptop's processor doing the work.
  • No subscription costs: Once downloaded, use it as much as you want, forever.
  • Instant responses: No waiting for server queues or network latency.

‚è∞ Understanding Knowledge Cutoff

Local AI models have a training cutoff date - they only know information up until they were trained. For example:

  • LFM2 was trained in late 2024 - it won't know about events from 2025
  • They can't search the web to get current information
  • The upside? This air-gapped limitation is actually a privacy feature - no search queries means no data leaks!

For current events or recent information, use cloud AI. For everything else, local AI's privacy advantage often outweighs having slightly older knowledge.

For the most demanding tasks, use cloud AI. Many professionals use all three - choosing the right tool for each job!

Troubleshooting

Model won't load?

  • Check you have enough disk space (need 2-3GB free)
  • Try restarting LM Studio
  • Check your laptop meets minimum requirements (4GB+ RAM recommended)
  • Make sure the model finished downloading completely

Responses are very slow?

  • LFM2 is designed to be very fast - if it's slow, your laptop may be running other programs
  • Close other applications while using LM Studio
  • Keep prompts shorter for faster responses
  • Make sure your laptop isn't in power-saving mode

Getting strange responses?

  • First, try starting a new chat (click the + button) - this fixes most issues!
  • Make your prompts more specific
  • Try rephrasing your question
  • Check if you have a system prompt set - look for the "Add Instructions" button at top right and clear any text there

Can't find the "Add Instructions" button?

  • Look at the top right of the chat window
  • It's near where you see "Model loaded" or the model name
  • Try making your LM Studio window bigger if the button is hidden
